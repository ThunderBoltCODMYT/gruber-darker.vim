# 🤝 Contributing

I welcome contributions from the community. Please:

1. **Report Issues**: Use GitHub Issues for bug reports and feature requests
2. **Submit Pull Requests**: Ensure code follows existing patterns and includes documentation
3. **Test Changes**: Verify compatibility across Vim 8.0+ and Neovim 0.5+
4. **Update Documentation**: Maintain consistency in help files and README
